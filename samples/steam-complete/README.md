# steam-complete

Steam integration sample for WebView2AppHost.

## Requirements

- Steam client running and signed in
- `Facepunch.Steamworks.Win64.dll` and `steam_api64.dll` next to the host executable
- this folder copied as `www/`

## What it demonstrates

- Steam initialization and overlay helpers
- achievements and stats
- Steam Cloud operations
- leaderboard workflows
- screenshot registration

This sample is intentionally opinionated and is meant as a full integration reference rather than a minimal demo.
