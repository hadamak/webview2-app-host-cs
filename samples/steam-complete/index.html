Steam UI

フレンド一覧、実績一覧、設定などの Steam 側 UI を開きます。WebView2 の画面上に重なる表示ではありません。

WebView2 画面上に重なるオーバーレイではなく、Steam 側の UI を開きます。

Achievements

実績は「この条件を達成した」という Steam 上の記録です。ここには Steamworks で登録した実績 API 名を入れます。

User Stats

User Stats はプレイ回数や累計スコアのような数値記録です。実績と違い、数値を読み書きできます。stat 名と型は Steam 側定義に合わせてください。

DLC / Ownership / Presence

本編や DLC の所有状態を確認したり、フレンド一覧に表示される現在状況を設定したりします。

Steam Cloud

セーブデータや設定ファイルを Steam 経由で同期します。ここではテキストとして JSON を保存する例を試せます。

Leaderboards

ハイスコアやタイムアタック用のランキング機能です。先に leaderboard を見つけるか作って、その handle を使ってスコアを送信します。

Auth / Screenshot

Web API 用の認証チケット取得と、Steam スクリーンショットの送信確認に使います。

Log

各 API 呼び出しの結果を表示します。失敗時は `reason` も確認してください。
