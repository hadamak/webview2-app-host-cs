AppBridge API デモ

ウィンドウ操作

AppBridge.requestFullscreen() / AppBridge.exitFullscreen() でフルスクリーンを制御します。
F11 キーでも同じ動作をします。ESC でフルスクリーン解除。

ウィンドウモード

ウィンドウタイトル変更

<title> タグの内容がリアルタイムでウィンドウタイトルバーに反映されます。
SPA のルーティングや状態変化に合わせてタイトルを更新できます。

localStorage 永続化

通常のブラウザと同じ localStorage API が使えます。
EXE 名から自動決定したユーザーデータディレクトリに保存されるため、
EXE を移動しても・再起動しても値が保持されます。

—

visibilitychange

ウィンドウの最小化・他アプリへの切替で visibilitychange イベントが発火します。
ブラウザ標準の document.visibilityState も同時に更新されるため、
BGM の停止・再開やゲームのポーズなどを通常のブラウザと同じコードで実装できます。
