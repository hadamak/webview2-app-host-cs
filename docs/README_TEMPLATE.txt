アプリケーションの利用ガイド

このアプリケーションを快適にご利用いただくためのガイドです。

1. 動作環境

このアプリを実行するには、以下の環境が必要です。

- OS: Windows 10 以降
- ランタイム: 
  - .NET Framework 4.7.2～4.8.1（Windows に標準搭載されています）
  - Microsoft Edge WebView2 ランタイム（Windows11:標準搭載、Windows10:推定の場合あり）

注意:
アプリ起動時に「WebView2 の初期化に失敗しました。」というエラーが出る場合は、以下の WebView2 ランタイムのダウンロードページから「Evergreen Bootstrapper」をダウンロードしてインストールしてください。
https://developer.microsoft.com/microsoft-edge/webview2/

2. 設定のカスタマイズ

EXE と同じ場所に user.conf.json を作成することで、ウィンドウの表示設定を変更できます。
設定した値はアプリ内の既定値より優先されます。記述しない項目はアプリの既定値が使われます。

[設定項目 : 説明]
- width / height: 起動時のウィンドウサイズ（ピクセル）
- fullscreen: 起動時にフルスクリーンにするかどうか (true / false)

例:
{
  "width": 1280,
  "height": 720,
  "fullscreen": false
}

3. トラブルシューティング

- 設定が反映されない: 設定ファイルを書き換えた後は、アプリを一度終了して再起動してください。
