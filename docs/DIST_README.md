# アプリケーションの利用ガイド

このアプリケーションを快適にご利用いただくためのガイドです。

## 1. 動作環境

このアプリを実行するには、以下の環境が必要です。

- **OS**: Windows 10 以降
- **ランタイム**: 
  - .NET Framework 4.7.2（通常、Windows に標準搭載されています）
  - **Microsoft Edge WebView2 ランタイム**

> [!TIP]
> アプリ起動時にエラーが出る場合は、[WebView2 ランタイムのダウンロードページ](https://developer.microsoft.com/microsoft-edge/webview2/) から「Evergreen Bootstrapper」をダウンロードしてインストールしてください。

## 2. 操作方法

### キーボードショートカット

| キー | 動作 |
|:---:|---|
| **F11** | フルスクリーン表示の切り替え |
| **ESC** | フルスクリーン表示の解除 |

## 3. 設定のカスタマイズ

アプリと同じ場所にある設定ファイル（`app.conf.json` 等）を編集することで、ウィンドウの初期サイズなどを変更できる場合があります（※アプリの構成によります）。

| 設定項目 | 説明 |
|---|---|
| `title` | ウィンドウのタイトル |
| `width` / `height` | 起動時のウィンドウサイズ（ピクセル） |
| `fullscreen` | 起動時にフルスクリーンにするかどうか (`true` / `false`) |

## 4. トラブルシューティング

- **画面が真っ白のまま進まない**: ネットワーク接続が不要なアプリであっても、WebView2 ランタイムが正しくインストールされていないと表示されない場合があります。上記「動作環境」を確認してください。
- **設定が反映されない**: 設定ファイルを書き換えた後は、アプリを一度終了して再起動してください。
