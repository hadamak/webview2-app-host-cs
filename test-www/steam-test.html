Steam プラグイン テスト（GenericDllPlugin / Facepunch.Steamworks）

初期化

SteamClient.Init(appId, asyncCallbacks) を呼び出します。 asyncCallbacks=false のとき SteamClient.RunCallbacks() を定期的に呼ぶ必要があります。

ユーザー情報

SteamClient.Name / SteamClient.SteamId はプロパティ。 メソッドとして呼び出すと、ディスパッチャがメソッド未発見時にプロパティ getter へフォールバックする。

実績 (Achievement 構造体)

Achievement は Steamworks.Data.Achievement 構造体。 Achievement.Create("ACH_NAME") でインスタンスを生成し、返ってきたプロキシ経由でメソッドを呼ぶ。

Stats (SteamUserStats)

SetStat は int/float オーバーロードを持つ。JS の number 型から C# のオーバーロードを自動解決する。

Steam クライアント UI (SteamFriends)

Steam 本体が別ウィンドウを開く動作。WebView2 の画面に重畳するオーバーレイではない。

スクリーンショット

SteamScreenshots.TriggerScreenshot は DirectX フック前提のため WebView2 では動作しない。
Host.Internal.WebView.CapturePreview() で RGB データを取得し、WriteScreenshot() に渡す。

Steam Cloud (SteamRemoteStorage)

FileWrite(name, byte[]) — JS 側は Array.from(Uint8Array) で渡す。
FileRead(name) → byte[] を数値配列で返す。TextDecoder でデコード可能。
