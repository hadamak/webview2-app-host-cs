Steam ブリッジ テスト（Facepunch.Steamworks / Proxy パススルー）

ユーザー情報

SteamClient の静的プロパティをプロパティ getter フォールバックで取得

実績 (Achievement 構造体)

Achievement は静的クラスでなく構造体。
Steam.Achievement.Trigger("ACH_NAME") → new Achievement("ACH_NAME").Trigger() として実行される。

Stats (SteamUserStats)

Facepunch では SetStatInt ではなく SetStat(string, int) のオーバーロード。
JS の number は double で届くが、ディスパッチャが int オーバーロードに型変換する。

オーバーレイ (SteamFriends)

WebView2 上への重畳は不可。Steam の該当 UI ウィンドウが開く（旧仕様と同じ動作）。
OpenOverlay(string type) と OpenWebOverlay(string url) の2種類。

スクリーンショット

C# 側で WebView2 をキャプチャし RGB byte[] を WriteScreenshot に直接渡す。

Steam Cloud (SteamRemoteStorage)

FileWrite(string name, byte[] data) — JS の Uint8Array は C# 側で byte[] に変換される。
FileRead(string name) → byte[] を数値配列で返す。TextDecoder でデコード可能。
ファイルサイズ確認は FileSize(string name)（GetFileSize ではない）。
